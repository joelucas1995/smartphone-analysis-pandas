# smartphone-analysis-pandas
# 📊 Smartphone Analysis com Pandas

Este repositório contém um script em **Python/Pandas** para análise exploratória de dados de smartphones (abril/2026).  
O objetivo é demonstrar boas práticas de **tratamento de dados** e **análises exploratórias** sem incluir o dataset original nem arquivos `.ipynb`.

---

## 🚀 Funcionalidades do script

1. **Descrição inicial dos dados**
   - Tipos de variáveis
   - Valores nulos
   - Valores únicos por coluna

2. **Tratamento de qualidade dos dados**
   - Remoção de colunas constantes
   - Remoção de linhas com valores nulos
   - Reset de índice

3. **Análises exploratórias (5 perguntas respondidas)**
   - Média de preço por categoria
   - Marcas com mais modelos listados
   - Distribuição dos sistemas operacionais
   - Média de capacidade de bateria por categoria de preço
   - Modelos com mais de 3 câmeras traseiras
